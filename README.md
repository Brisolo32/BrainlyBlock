# Como Instalar:

1. Baixe o crx da página de _releases_
2. Abra o chrome (se não abriu)
3. Vá para `chrome://extensions`
4. Arraste o crx até a página
5. Clique **Adicionar Extensão**
6. Pronto, Você instalou!
